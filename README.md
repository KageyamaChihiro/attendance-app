# 勤怠管理アプリ

## 概要

Laravelを使用して作成したシンプルな勤怠管理アプリです。
出勤・退勤の打刻、勤怠一覧の表示、勤務時間の編集機能を実装しています。

---

## 機能一覧

* 出勤打刻
* 退勤打刻
* 勤怠一覧表示
* 勤務時間編集
* 勤務時間の自動計算（出勤〜退勤）

---

## 使用技術

* PHP（Laravel）
* MySQL
* Docker（Laravel Sail）
* Blade
